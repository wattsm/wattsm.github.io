About

I am a software engineer living and working in Sheffield, England. I love programming (particularly in F#), science, history and gaming among other things.

This blog is primarily about software development, with occasional forays into science, history and gaming. The blog name comes from the first part of the throne name of Ramesses II, Usermaatre Setepenre, and also a song by the excellent Nile.

GitHub

I have various repositories on GitHub, with the most interesting / useful listed below.

  • Slumber - an F# framework for creating REST services.
  • Elite:Dangerous Rares - using an F# DBSCAN implementation to cluster rare goods providers in Elite:Dangerous (related post here).
  • HandyFS - a collection of handy F# utility functions.
  • ConsoleApp - a small F# library that can be used to quickly create command line apps.
